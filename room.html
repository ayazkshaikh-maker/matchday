One more step

Add your Supabase keys to enable live rooms.

  1. Create a free project at supabase.com
  2. Open Project Settings → API and copy the Project URL and the anon public key
  3. Paste them into the CONFIG block at the top of room.html
  4. Redeploy. Realtime is on by default — no database tables needed.
